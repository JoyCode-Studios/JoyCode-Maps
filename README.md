# JoyCode Maps

JoyCode Maps is a simple, branded web mapping application built with [Leaflet.js](https://leafletjs.com/) and [OpenStreetMap](https://www.openstreetmap.org/).  
It provides a clean interface with search, geolocation, and shareable links using query parameters.
